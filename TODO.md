# Upkeep Feature Roadmap

## Complete

- **Filter on Updates tab** - Consistency with Installed tab
- **Orphan packages tab** - Packages no longer needed as deps
- **Package search tab** - Search across repos + AUR, async/non-blocking
- **Package info pane** - Toggle with `?`, async fetching
- **Arch news viewer** - RSS feed with manual intervention warnings
- **Dependency view** - Show what depends on selected package
- **Cache cleanup** - Clear old package versions
- **Update confirmation** - Preview before running destructive actions
- **Export package list** - Backup to files or clipboard
